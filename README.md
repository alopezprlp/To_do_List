## Description

This is a simple todo list made with Express, Knex, Objection and Reactjs

## Server Configuration

Set enviroment .env

```bash
DATABASE_URL=postgres://username:password@host:port/db_name
```

## Server installation

```bash
$ cd backend
$ npm install
```

## Running the app

```bash
$ npm start
```
## Client installation

```bash
$ cd frontend
$ npm install
```

## Running the app

```bash
$ npm start
```
